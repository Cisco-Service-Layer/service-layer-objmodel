Enum SlLabelAction

MPLS label actions.

Namespace: ServiceLayer
Assembly: ServiceLayer.dll
Syntax
public enum SlLabelAction

Fields

Name Description
PopAndForward

Pop and forward. 0x2 Pop outer label and forward. Used for per CE and per prefix label disposition at egress PE.
PopAndLookup

Pop and lookup. 0x5 Pop and lookup the next header in the packet (MPLS, IPv4, IPv6) in the specified Vrf. Used for per vrf labels at egress PE.
PopAndLookupIpv4

Pop and lookup IPv4. 0x3 Pop and lookup the IPv4 next header in the specified Vrf. Used for per vrf labels at egress PE.
PopAndLookupIpv6

Pop and lookup IPv6. 0x4 Pop and lookup the IPv6 next header in the specified Vrf. Used for per vrf labels at egress PE.
Push

Push. 0x6 Push MPLS labels on a IP packet. Support only exists for IP imposition.
Reserved

Reserved. 0x0
Swap

Swap. 0x1 Swap outer label and forward. Used at tunnel midpoint for swapping labels.
